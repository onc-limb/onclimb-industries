# Coding Rules Templates

コーディング規約を作成するためのテンプレート集。

## 使い方

1. `architecture.template.md` または `languages.template.md` をコピーする
2. プロジェクトに合わせて各セクションを埋める
3. 完成したものを `standards/coding-rules/` に配置するか、プロジェクトリポジトリに含める

## テンプレート一覧

| テンプレート | 用途 |
|---|---|
| [architecture.template.md](architecture.template.md) | アーキテクチャ規約（ルール、判断ガイド、適用例） |
| [languages.template.md](languages.template.md) | 言語別規約（命名規則、実装パターン、テストの書き方等） |

## 記入例

`standards/coding-rules/` に実際に記入した例があります。

- `architecture/clean-architecture.md` — クリーンアーキテクチャ規約
- `architecture/layered-architecture.md` — レイヤードアーキテクチャ規約
- `languages/typescript.md` — TypeScript コーディング規約
- `languages/go.md` — Go コーディング規約
- `languages/python.md` — Python コーディング規約
