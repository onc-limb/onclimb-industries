# 家計カテゴリ定義

エントリ（変動費）・固定費に付けるカテゴリの正規リストと判定基準。
**scripts/aggregate.py の `KNOWN_CATEGORIES` 定数と必ず同期させる**（更新は本人確認のうえ手動で両方を直す）。

## カテゴリ一覧

| カテゴリ | 主な用途 | 例 | 備考 |
|---|---|---|---|
| 食費 | 食材・外食・カフェ・中食 | スーパー、コンビニ弁当、レストラン | 酒類も含める **(ASSUMPTION)** |
| 日用品 | 消耗品・生活雑貨 | 洗剤、ティッシュ、電池、キッチン用品 | |
| 交通 | 電車・バス・タクシー・ガソリン | Suica チャージ（乗車利用分）、タクシー | 私用の移動のみ。案件の交通費は事業側（tax-prep-organizer） |
| 医療 | 病院・薬 | 診察代、処方薬、市販薬 | |
| 交際 | 他者との飲食・贈り物 | 飲み会、冠婚葬祭、プレゼント | 1 人の外食は食費 |
| 趣味・娯楽 | 娯楽・レジャー・私的な書籍 | 映画、ゲーム、旅行、漫画 | 技術書は原則事業側の可能性を指摘 |
| 衣服・美容 | 衣類・理美容 | 服、靴、散髪 | |
| 教育 | 私的な学習 | 語学教室、私的な講座 | 業務直結の学習費は事業側の可能性を指摘 |
| 住居 | 家賃・住宅関連 | 家賃、共益費、更新料 | 主に固定費で使用 |
| 水道光熱 | 電気・ガス・水道 | 電気代、ガス代、水道代 | 主に固定費で使用 |
| 通信 | 回線・携帯 | 光回線、スマホ料金 | 主に固定費で使用 |
| 保険 | 保険料 | 生命保険、火災保険 | 主に固定費で使用 |
| サブスク | 定額サービス | 動画配信、音楽、クラウドストレージ | 主に固定費で使用。開発ツールの課金は事業側の可能性を指摘 |
| 特別支出 | 高額・非経常の支出 | 家電の買い替え、引越し費用 | 目安: 3 万円以上かつ非経常 **(ASSUMPTION)** |
| その他 | 上記に当てはまらない・確証がないもの | | 迷ったらここ + memo に理由を書く |

## 判定の原則

1. **確証がなければ「その他」**に倒し、`memo` に迷った理由を書く。ユーザー確認時に修正してもらう。
2. **店名だけで決めつけない**。総合スーパー・ドラッグストア・Amazon 等は品目が読めれば品目で判定し、
   読めなければ「その他」+ memo（例: `memo: "ドラッグストア。食品か日用品か不明"`）。
3. **1 レシート複数カテゴリ**は、明細が読めて分割合計がレシート合計と一致する場合のみ分割してよい。
   一致しない・読めない場合はレシート単位で主たる用途のカテゴリ 1 つに寄せる。
4. **事業経費らしきもの**（技術書・サーバ代・開発ツール・案件の交通費など）は本スキルに取り込まず、
   `ultron-tax-prep-organizer` での整理を案内する。ユーザーが「私用」と明言した場合のみ家計側に入れる。

## payment_method の値

`cash` / `credit` / `e-money`（交通系 IC・QR 決済含む **(ASSUMPTION)**）/ `bank_transfer` / `unknown`
