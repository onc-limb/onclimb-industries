## 業種の調べ方（登録時）

CSV に業種が無いため、銘柄コードから公開情報で確認して `sector set` する。信頼できる出典:

- **Yahoo!ファイナンス** `finance.yahoo.co.jp/quote/<コード>.T` の「業種」
- **株探（かぶたん）** `kabutan.jp/stock/?code=<コード>`
- **バフェット・コード** `buffett-code.com/company/<コード>`（33業種を明記）
- **会社四季報オンライン** / 各証券会社の銘柄ページ

判断が割れやすい例（本ポートフォリオでの確定値）:

- ネットメディア（アイティメディア 2148）→ **サービス業**（情報・通信業ではない）
- ソフトウェア開発（トランヴィア 545A / ＣＡＣＨＤ / ＰＣＩＨＤ）→ **情報・通信業**
- 人材・研修（ウィルグループ / パーソル / メイテック / インソース / エン / キャリアリンク）→ **サービス業**
- 電機・機械商社（サンワテクノス / 兼松 / カノークス）→ **卸売業**（電気機器・鉄鋼ではない）
- 自動車部品でもホース・ベルト・ゴム主体（ニチリン / 三ツ星ベルト / 藤倉コンポジット）→ **ゴム製品**
- 自動車ディーラー（ＶＴＨＤ）→ **小売業**（輸送用機器ではない）

確信が持てない中小型株は必ず Web で裏取りし、推測で入れた場合はレポートで「要確認」と伝える。

## 補足: 東証17業種

より粗い 17 業種でまとめたい場合は `sector set --sector17 "<17業種>"` で任意に併記できる
（集計は 33 業種ベース）。17 業種は「電機・精密」「機械」「素材・化学」「情報通信・サービスその他」
「商社・卸売」「小売」「銀行」「金融（除く銀行）」「不動産」「建設・資材」「食品」「医薬品」
「自動車・輸送機」「鉄鋼・非鉄」「エネルギー資源」「電力・ガス」「運輸・物流」。
