# Claude レビューのチェックリスト（参照メモ）

生成したおすすめリストの末尾に必ず添える「Claude のレビュー」で見る観点。
**投資助言ではなく情報整理の講評**であることを忘れない。観点は運用で追記してよい（自己進化の対象）。

## 必ず触れる観点

1. **偏り**
   - 業種(セクター)の集中はないか（同一業種に偏ると分散が効かない）。
   - 利回り帯の偏り（**超高利回り(例 6%超)に寄っていないか** = 減配・タコ足配当リスクが上がる）。
2. **要再確認の銘柄**（合格でも注意を促す）
   - 配当性向が閾値ギリギリ（例 45〜49%）の銘柄。
   - 特別配・記念配で利回りが膨らんでいる疑いのある銘柄。
   - 営業利益の伸びが鈍化/横ばいで、増配余地が細い銘柄。
   - データの一部が「未取得(insufficient)」で判定根拠が薄い銘柄。
3. **データの鮮度・出所**
   - 取得日を明記。四半期跨ぎ・決算発表直後は数値が動く点に注意喚起。
   - 各銘柄に出所 URL があるか（無い値は載せない/未取得と明記）。
4. **免責**（必須・毎回）
   - 公開情報の整理であって投資助言ではない / 最終判断は自己責任 / 発注前に一次情報(IR・有報)で再確認。

## 書き方のトーン

- 断定的な推奨（「買うべき」等）を避け、「候補」「〜の傾向」「要確認」という整理の語彙に統一する。
- 合格銘柄を並べて終わりにせず、「で、何に注意して見ると良いか」を一段加える（このスキルの差別化点）。

## 進化メモ（運用で追記する欄）

- レビューで実際に効いた指摘・見落とした観点をここに残し、チェックリストを更新する。
