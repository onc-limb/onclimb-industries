# 勘定科目対応表（フリーランスエンジニア向け）

分類候補を付けるときの参照表。**これは税務助言ではなく、一般的な整理の目安**であり、
最終的な科目の確定は本人 / 税理士が行う。迷ったら科目を確定せず「要確認」に分離する。

- 根拠: 国税庁の青色申告決算書（一般用）に記載される標準的な経費科目区分を下敷きに、
  フリーランスエンジニアの頻出取引を対応付けたもの。**税法の解釈・節税判断は含まない**。
- この表の更新は、本人 / 税理士の確認を経た**手動追記のみ**とする（自動改訂しない）。

## 収入

| 勘定科目 | 典型的な取引例 |
|---|---|
| 売上高 | 業務委託報酬（準委任 / 請負）、SES 精算、受託開発の検収金額 |
| 雑収入 | アフィリエイト・技術記事の原稿料など事業付随の少額収入 **（事業規模なら売上高側の検討 → 要確認）** |

## 経費（頻出順の目安）

| 勘定科目 | 典型的な取引例 | 注意（要確認になりやすい点 |
|---|---|---|
| 通信費 | インターネット回線、携帯電話料金、レンタルサーバー・ドメイン、クラウド利用料（AWS/GCP 等）※ | 自宅回線・携帯は**家事按分**が前提。※クラウド利用料を通信費とするか支払手数料等とするかは運用が分かれる → 初回は要確認 |
| 消耗品費 | キーボード・マウス・ケーブル、10 万円未満の PC・モニター、文房具 | **10 万円以上**の機材は減価償却（工具器具備品）の検討 → 要確認 |
| 新聞図書費 | 技術書、電子書籍、技術雑誌、有料技術情報の購読 | 学習サービス（Udemy 等）を新聞図書費とするか研修費とするかは運用が分かれる → 要確認 |
| 旅費交通費 | 客先・勉強会への電車代、出張の宿泊費・航空券 | 私的移動との混在に注意（IC カード履歴は事業分のみ抽出） |
| 地代家賃 | 自宅兼事務所の家賃、コワーキングスペース月額 | 自宅家賃は**家事按分必須**。按分率は本人が確定した値のみ使用 → 未確定なら要確認 |
| 外注工賃 | デザイン・コーディングの外注、業務の再委託 | 源泉徴収の要否は判断しない（支払調書関連は税理士へ） |
| 水道光熱費 | 自宅兼事務所の電気代 | **家事按分必須** → 率が未確定なら要確認 |
| 支払手数料 | 銀行振込手数料、決済手数料、エージェント手数料、SaaS 利用料※ | ※開発用 SaaS（GitHub, IDE, AI ツール等）の科目は通信費と分かれがち → 初回は要確認 |
| 研修費 | 有料セミナー・カンファレンス参加費、オンライン講座 | 決算書の標準科目にない場合は雑費・新聞図書費等との整理 → 要確認 |
| 接待交際費 | 取引先との会食、手土産 | 1 人あたり金額や目的の記録が必要。私的飲食との区別 → 迷えば要確認 |
| 会議費 | 打ち合わせのカフェ代・会議室利用料 | 接待交際費との線引きが曖昧な支出 → 迷えば要確認 |
| 広告宣伝費 | 自サイトの広告出稿、名刺・ポートフォリオ制作 | |
| 租税公課 | 個人事業税、収入印紙、自動車税（事業按分分） | **所得税・住民税は経費にならない** → 混入していたら要確認 |
| 損害保険料 | 賠償責任保険（フリーランス向け保険等）、事務所の火災保険 | 生命保険料は経費でなく所得控除側 → 要確認 |
| 修繕費 | PC・機材の修理費 | 資本的支出との区分が問題になる高額修理 → 要確認 |
| 減価償却費 | 10 万円以上の PC 等の償却費 | 償却計算そのものは本スキルの対象外（会計ソフト / 税理士へ）。対象資産の購入は要確認に分離 |
| 荷造運賃 | 機材返送・納品物の送料 | |
| 雑費 | 上記いずれにも当てはまらない少額経費 | 雑費が膨らむのは分類不全のサイン。多用しない → 迷えば要確認 |

## 「要確認」に必ず分離する取引（科目を確定しない）

1. **家事按分が絡み、按分率が未確定**のもの（家賃・電気代・通信費・車両関連）。
2. **10 万円以上**の資産購入（減価償却 vs 消耗品費 / 少額減価償却資産特例の判断が必要）。
3. **私用と事業用が混在**する購入（EC サイトのまとめ買い等）。
4. **科目の運用が分かれる**もの（クラウド利用料・SaaS・学習サービスの初回分類）。
5. 摘要から**内容が特定できない**取引。
6. 収入のうち事業所得か雑所得か判断が要るもの。
